about Alight Digital Architecture

UI and server-side logic is decoupled.
UI “widgets” are Angular components managed and built independently.
UI app is an Angular app, Business logic and data access is performed by uServices behind well-defined APIs
uServices are discrete services organized by function, and are independent of each other’s code base and configuration settings.
Much faster build times. Smaller deployments.
Each uService is responsible for embedding its own http engine (e.g. tomcat for Spring Boot, express for nodejs, and so on). Standards/guidance provided by architecture teams.
uServices are packaged into docker images and are deployed independent from other uService images.
A single uService is deployed at a time.
